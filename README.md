# InstaPro

InstaPro — это учебный аналог популярной социальной сети для обмена фотографиями. Проект включает как базовую функциональность, повторяющую конкурентов, так и дополнительные улучшения интерфейса, делающие использование более удобным и приятным.

---

## Реализованная основная функциональность

- Интеграция верстки списка постов с API;
- Реализация страницы добавления нового поста;
- Реализация страницы с постами конкретного пользователя;
- Реализация функционала лайков (счётчик, визуальное отображение, сохранение состояния).

### Временные оценки:

- **Первоначальная оценка:** ~4 дня (32 часа)
- **Фактически затраченное время:** ~6 дней (48 часов)

---

## Дополнительные задачи (улучшения интерфейса)

- **Кнопка удаления поста** — реализована возможность удаления своих постов;
- **Анимация лайка** — добавлена анимация при нажатии на лайк для более живого UX;
- **Отображение пользователей, поставивших лайк** — при наведении отображаются имена тех, кто лайкнул пост.

### Временные оценки:

- **Первоначальная оценка:** 6 часов
- **Фактически затраченное время:** 8 часов
